Dashboard utama web absensi

pengambilan foto serta pengambilan titik lokasi absensi

Saya membuat web absensi menggunakan framework Laravel dengan database MySQL yang diatur melalui XAMPP. Untuk styling, saya memakai Bootstrap, sementara untuk teknologi geolokasi dan webcam, saya menggunakan geolocation API dan webcam.js dari Google. Tampilan peta lokasi dalam aplikasi ini didukung oleh Leaflet.js untuk visualisasi yang interaktif dan informatif.

Dalam proyek inovatif ini, saya telah merancang solusi IoT yang revolusioner untuk meningkatkan efisiensi dan kenyamanan dalam merawat tanaman.
